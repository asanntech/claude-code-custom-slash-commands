# claude-code-custom-slash-commands

Claude Code用のカスタムスラッシュコマンド集

## コマンド一覧

### `/commit-and-push`
ブランチ作成からプッシュまでの完全なgitワークフローを自動化します。コミットメッセージは自動生成されます。

### `/create-github-issue`
Markdownファイルの内容を読み込み、MCPサーバー経由でGitHub Issueを作成します。

### `/implement-task`
Task Masterのタスク/サブタスクを実行します。タスク詳細の取得、実装、テスト、ステータス更新まで自動化します。

### `/tasks-to-github-issues`
Task Masterのtasks.jsonを読み込み、日本語に翻訳してGitHub Issuesとして作成します。
